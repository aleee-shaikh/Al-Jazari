About Al-Jazari

Al-Jazari-1 is a wireless, teleoperated service robot designed to assist with daily household tasks. Its primary goal is to help users by providing mobility support, load carrying, and home surveillance. The robot can carry items such as food, drinks, and other small loads, and is capable of autonomous patrolling within indoor environments.

From a technical standpoint, Al-Jazari-1 was built to be affordable, portable, and easy to control. It is powered by an Arduino Uno and controlled via Bluetooth (HC-05), with integrated DC motors enabling smooth movement, arm articulation, gripper control, and base rotation. The robot also features real-time video streaming viewable on a laptop and includes an interactive, animated AI presenter powered by Gemini LLM, capable of answering user queries in natural language.

Hardware Components

  • Arduino UNO R3 – Main microcontroller board
  • 9× DC Motors – For movement, arm lift, base rotation, and gripper control
  • Aluminum Frame – Lightweight and durable structural body
  • 12V Battery – Power source for motors and electronics
  • Android Mobile Device – Used for:
    • 1. Running the assistant app (Gemini LLM)
    • 2. Providing real-time video via mobile camera
  • HC-05 Bluetooth Module – For wireless control
  • Motor Driver Modules (e.g., L298N) – To control DC motors
  • Wheels For movement

Software Stack

  • C# with Emgu CV
    Used for real-time video streaming from the robot to a laptop, enabling remote monitoring and navigation.
  • HTML, JavaScript, and CSS
    Developed the front-end interface of the assistant app, providing a user-friendly experience for interaction.
  • Gemini LLM (via Web Integration)
    Powers the animated AI assistant, capable of answering voice or text-based queries in natural language.
